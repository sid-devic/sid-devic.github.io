# Siddartha Devic's Personal Website

This is my personal academic website hosted on GitHub Pages. The site is built using static HTML, CSS, and JavaScript - no build process required.

## Structure

- `index.html` - Main homepage with bio, publications, and research information
- `rejections.html` - List of academic rejections and application materials
- `art/` - Art gallery showcasing my digital art from 2016-2021
  - `index.html` - Gallery homepage with album overview
  - Individual year/quarter pages (2016.html, 2020-Q1.html, etc.)
  - `media/` - Image assets (large, small, and thumbnail versions)
  - `public/` - Gallery JavaScript libraries and CSS

## Assets

- `assets/css/style.css` - Main stylesheet
- `assets/js/scale.fix.js` - Responsive scaling script
- `assets/fonts/` - Noto Sans font files
- `assets/img/` - Profile images and icons
- `assets/files/` - PDF documents (CV, research statements, etc.)

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Art Gallery**: Interactive photo gallery with lightbox functionality
- **Dark Theme**: Optimized for dark mode viewing
- **Academic Focus**: Clean layout highlighting research and publications

## Deployment

This site is automatically deployed via GitHub Pages. Simply push changes to the `master` branch and they will be live at [sid.devic.us](https://sid.devic.us).
